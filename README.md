# dotfiles

Dotfiles on my mac. I'll create a branch for Ubuntu.

## 一、Install

Open a terminal window on your mac. Recommend Item2 and oh-my-zsh.

1. Install Homebrew
2. Install Item2
3. Install A Nerd Font
4. Install Nvim
5. Install Nevide
6. Install Ripgrep
7. Install Node
8. Install lazygit

## 二、Usage

```shell
cd dotfiles
```

Create a soft link for nvim

```shell
ln -s dotfiles/nvim ~/.config/nvim
```
